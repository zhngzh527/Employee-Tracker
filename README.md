# Employee Tracker
Module 12 Challenge SQL Employee Tracker
## Description
This is a command-line application from scratch to manage a company's employee database, using Node.js, Inquirer, and MySQL. This application allow business owners to view and manage the departments, roles, and employees in a company so that they can organize and plan their business.

## Build with
 - MySQL2 package
 - Inquirer package
 - console.table.package
